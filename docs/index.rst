Welcome to pommesinvest's documentation!
===========================================

**A bottom-up fundamental investment model
for the German electricity sector**

This is the documentation of ``pommesinvest``, the investment variant of the
fundamental power market model *POMMES* (**PO**\wer
**M**\arket **M**\odel of **E**\nergy and re **S**\ources).

If you are interested in the data preparation routine,
you can find a description `here <https://pommesdata.readthedocs.io/>`_.
The dispatch model can be found
`here <https://pommesdispatch.readthedocs.io/>`_.
